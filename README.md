## Overview

<a href="#">Lightroom Classic</a> delivers professional-grade photo editing through an interface designed for both simplicity and depth. The application serves professional photographers who need complete control over their workflow.

The <a href="#">map module</a> displays photos by location on interactive maps. View and organize images by where they were taken.

<a href="#">Face recognition</a> identifies people across your photo library automatically. Organize portraits by subject with AI assistance.

<a href="#">HDR and panorama</a> merge multiple exposures and images. Create high dynamic range and wide-angle images.

The <a href="#">local adjustments</a> apply edits to specific image areas. Use brushes, gradients, and radial filters for targeted editing.

<a href="#">Presets</a> save and apply consistent looks efficiently. Build custom presets or use included options.

<a href="#">Export options</a> save images with precise control over format and quality. Prepare files for any output requirement.

<a href="#">Plugin support</a> extends capabilities through third-party additions. Access additional export destinations and editing tools.

<a href="#">Photoshop integration</a> sends images for advanced editing seamlessly. Round-trip to Photoshop and back without complexity.

<a href="#">Keyword and metadata</a> management organizes images for searching. Build comprehensive metadata for efficient retrieval.

<a href="#">Backup catalogs</a> protect your editing work automatically. Safeguard the work invested in organizing and editing.
